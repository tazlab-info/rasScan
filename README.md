# rasScan
Python

j'ai utilisé python car c'est un langage simple, ce dont j'avait besoin, pour sa facilité de debugging et car openfoodfacts à une api python.

Données

Les données sont fournies par openfoodfacts (https://www.openfoodfacts.org).

pourquoi

Suite a un épisode marquant de ma vie, le vidage du frigo, j'ai constaté que la moissisure était peu utile mais tres odorante donc je me suis mis a creer un code pour evité cela a l'avenir.

comment

Pour l'utilisé hebergé le code python sur un rasberry pi 0 0w ou autre machine pouvait executé du python, puis fait un cron en mettant pour chaque demarrage de lancer le code ainsi que le server. (Réalisable automatiqument prochainement grace a un .sh) avec le html js et css.

Outil

Il vous faudra donc un scanner pour scannez les code-barres personnelement j'ai utilisé un "Colemeter" trainant dans un tirroir, pui je l'est programmé grace a se pdf (https://barcode.com.sg/wp-content/uploads/2020/02/EC103-User-Manual.pdf) pour ajouté un suffixe entrée (il simule la touche entré a chaque scanne).
